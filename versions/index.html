Version Log

This public version log records Agentic Accessibility vision, AWGS standard, and guideline versions. It is a publication history document, not a source of new conformance requirements.

AWGS Guidelines 1.1

Versioning Policy Notes

  • AWGS standard versions should not overwrite prior public standard versions without an archive path.
  • Normative changes should be made in the canonical standard source before presentation data is updated.
  • Informative guidance and knowledge base articles should cite the applicable AWGS version.
  • If future certification or assurance is ever considered, it should be versioned separately from current AWGS 1.0 maturity target statements unless the standard defines otherwise.
